Installation

The very first step is to download the Anibry library here. After downloading, there are different methods listed below to start using the classes.

Import the file:

Unzip the downloaded file and import it into your stylesheet

@import "anibry";

Or do this:

@import url( "anibry.css");

Link the file to your webpage:
You can also get access to the library by linking the CSS file to your webpage.

<head>
  <link
    rel = "stylesheet" >
    href = "anibry.css" />
  />
</head>
