Using JavaScript

Alternatively, you can make your animations even more dynamic by using JavaScript.
Create a Javascript file and link it to your html file appropriately. To make the process easy, you can just copy the code below and paste is inside the Javascript file you just created.
Next up is to change the "targetelement" in the JavaScript code to the ones you want to animate and the "animationname" to your desired animation classname.
That's it! 

                
  const targetClass = document.querySelector("targetelement");

  let targetClassName = targetClass.innerHTML;

    targetClass.addEventListener("click", function () {

      targetelement.className = "anibry";

      targetelement.classList.add(`${animationname}`);

    }
  });
}

Your animation is all set!
